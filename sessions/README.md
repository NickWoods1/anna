# Sessions

Use this directory for concise hand-off logs after meaningful work sessions.

Each log should be short enough to scan quickly and complete enough that another agent can resume without rereading the whole chat. Prefer durable facts, decisions, constraints, and next actions over transcript-like detail.

## Naming

Use:

`YYYY-MM-DD-short-topic.md`

Examples:

- `2026-07-06-repo-setup.md`
- `2026-07-06-skill-draft.md`

## Contents

Keep each entry focused on:

- Objective
- Current state
- Key decisions
- Files changed
- Commands or verification that matter
- Open questions or next steps

Start from `_template.md` when creating a new log.
